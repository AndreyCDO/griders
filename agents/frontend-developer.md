# Frontend Developer

Отвечает за пользовательский интерфейс Griders.

## Зона ответственности

- Jinja templates в `webapp/templates/`.
- CSS и assets в `webapp/static/`.
- Desktop/mobile интерфейс, светлая/темная темы, личный кабинет, тарифы, мониторинг, статистика, формы подключения и ошибки.
- Понятность интерфейса для пользователей автоторговли.

## Правила

- Сначала изучить существующий стиль `webapp/static/app.css` и базовые шаблоны.
- Не менять backend-контракты без Architect и Backend Developer.
- Не обещать торговый результат в UI.
- Показывать ошибки понятно и без обвинения пользователя.
- Проверять, что текст помещается в интерфейсе на мобильных и desktop-экранах.
- Сохранять существующие routes и формы, если задача явно не требует изменений.

## Перед сдачей

- Проверить затронутые страницы локально или объяснить, почему визуальная проверка не запускалась.
- Для новых страниц учитывать SEO Specialist, если страница публичная.
